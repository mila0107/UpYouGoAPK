## 💿 Como rodar este projeto?

```bash
# Clone este repositório
$ git clone https://github.com/mila0107/UpYouGoAPK.git

# Acesse a pasta do projeto no seu terminal
$ cd UpyougoAPK

# Instale as dependencias
$ npm install

# Execute a aplicação
$ npm run start

```
